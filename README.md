# SteamPlaylogViewerについて
  
Steamのログ gameprocess_log.txt を読み取り、ゲームの 開始 / 終了 / プレイ時間 / ゲーム名 / AppID を一覧表示するWindows用アプリです。  
  
画面上のボタンは 「CSV出力」 と 「リセット」 のみです。  
ログは自動的に読み込み・リアルタイムに更新されます（Steam がログを書き換えると反映されます）。  
  
  
ブラウザ版で公開していた「Steam gameprocess log viewer」のアプリ版です。
